Number Guessing Game in Python

Introduction

This simple Python script is a Number Guessing Game where the computer picks a random number between 1 and 10, and the player must guess it. The program provides hints (too high or too low) until the correct number is guessed.

This script demonstrates the use of loops, conditional statements, and user input handling, making it a great beginner-friendly project.

Python
Copy code

import random


# Generate a random number between 1 and 10

secret_number = random.randint(1, 10)


print("Welcome to the Number Guessing Game!")

print("I have chosen a number between 1 and 10. Can you guess it?")


# Loop until the user guesses correctly

while True:

    guess = int(input("Enter your guess: "))


    if guess < secret_number:

        print("Too low! Try again.")

    elif guess > secret_number:

        print("Too high! Try again.")

    else:

        print(f"Congratulations! The correct number was {secret_number}.")

        break

  • The randint(1, 10) function generates a random integer between 1 and 10.
  • This number is stored in secret_number, which the player has to guess.

print("Welcome to the Number Guessing Game!")

  • The input() function takes the player's guess, and int() converts it into a number.

How the Game Works

  1. The program picks a random number between 1 and 10.
  2. The player enters a guess using input().
  3. If the guess is too high or too low, the game provides hints.
  4. The player keeps guessing until they find the correct number.
  5. Once guessed correctly, the game congratulates the player and ends.

This project is a great way to practice loops, conditions, and user input handling in Python. You can improve it by adding features like score tracking, difficulty levels, or a limited number of guesses!
